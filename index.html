Pre-commit

Forget about Cmd-U

Download this project as a .zip file Download this project as a tar.gz file

CocoaPod that automatically installs a git pre-commit hook from the working directory whenever you run pod install.

Quick Start

  1. Add pod "pre-commit" to Podfile.
  2. mkdir -p .hooks/ && echo "exit 0" > .hooks/pre-commit
  3. pod install

Recommended uses

  • Run unit tests or analyzer via xctool
  • Run OCLint
  • Ensure commit message and author are up to spec.
